# 🧾 Faktura Bot — Telegram-бот для обработки фактур

Telegram-бот, который принимает фото или PDF фактур, распознаёт текст через OCR.space API, извлекает данные о товарах и записывает в Google Sheets с расчётом налога, маржи и новых цен.

## 🚀 Возможности

- 📸 Приём фото и PDF фактур
- 🔍 OCR-распознавание текста (польский язык)
- 📊 Автоматическое извлечение данных: дата, номер фактуры, товары, цены, количество
- 💰 Расчёт: налог, маржа, новая цена
- 📋 Запись в Google Sheets (создание новой таблицы или добавление к существующей)
- ⚙️ Настройка налога и маржи через кнопки бота
- 🔘 Удобное inline-меню (без необходимости вводить команды)

## 📦 Установка

### 1. Клонируйте репозиторий

```bash
git clone <repo-url>
cd faktura_bot
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

```bash
cp .env.example .env
```

Отредактируйте `.env` и заполните:

| Переменная | Откуда получить |
|---|---|
| `TELEGRAM_BOT_TOKEN` | У [@BotFather](https://t.me/BotFather) в Telegram |
| `OCR_API_KEY` | На [ocr.space/ocrapi/freekey](https://ocr.space/ocrapi/freekey) |
| `GOOGLE_CREDENTIALS_FILE` | Смотри раздел «Настройка Google Sheets» ниже |

### 5. Настройка Google Sheets API

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект (или используйте существующий)
3. Включите **Google Sheets API** и **Google Drive API**:
   - API & Services → Library → найдите и включите оба API
4. Создайте Service Account:
   - API & Services → Credentials → Create Credentials → Service Account
   - Дайте имя (например `faktura-bot`)
   - Нажмите Done
5. Создайте ключ:
   - Нажмите на созданный Service Account
   - Keys → Add Key → Create New Key → JSON
   - Скачайте файл и сохраните как `credentials.json` в папку проекта

> ⚠️ **Важно**: `credentials.json` содержит секретные данные. Не загружайте его в git!

### 6. Запуск бота

```bash
python bot.py
```

## 📊 Структура таблицы Google Sheets

| Дата покупки | Номер фактуры | Название товара | Цена закупки (шт.) | Количество | Общая стоимость | Налог (%) | Маржа (%) | Новая цена (шт.) | Цена всех |
|---|---|---|---|---|---|---|---|---|---|

### Формулы:
- **Общая стоимость** = Цена закупки × Количество
- **Новая цена** = Цена закупки + (Цена закупки × Налог%) + (Цена закупки × Маржа%)
- **Цена всех** = Новая цена × Количество

## 🤖 Использование бота

Отправьте боту фото или PDF фактуры. Бот автоматически:
1. Распознает текст
2. Извлечёт данные о товарах
3. Рассчитает новые цены
4. Запишет в Google Sheets
5. Отправит вам сводку со ссылкой на таблицу

### Кнопки меню:
- 📸 **Загрузить фактуру** — инструкция по отправке
- 📊 **Открыть таблицу** — ссылка на Google Sheets
- ⚙️ **Настройки** — изменить налог и маржу
- ❓ **Помощь** — справка по использованию
